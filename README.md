# יומן סופגניות 🍩 (RTL + Themes + שיתוף משפחתי)

זו אפליקציית Next.js + Supabase (חינמי) לניהול דירוגי סופגניות:
- התחברות/הרשמה
- יומן אישי + פיד משפחתי משותף
- יצירת משפחה עם **קוד קצר** להצטרפות
- ממשק בעברית מימין לשמאל
- ערכות נושא: בהיר / כהה / חנוכה

## דרישות
- Node.js 18+ (מומלץ 20+)
- חשבון Supabase (חינמי)

## התקנה והפעלה
1) פתחי פרויקט חדש ב-Supabase.
2) הריצי את ה-SQL שנמצא כאן:
   `supabase/schema.sql`

3) צרי קובץ `.env.local` לפי `.env.example`:
```bash
cp .env.example .env.local
```

4) מלאי:
- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`

(את מוצאת אותם ב-Supabase: Project Settings → API)

5) התקיני והריצי:
```bash
npm install
npm run dev
```

פתחי: http://localhost:3000

## איך השיתוף עובד
- לכל רשומה יש `visibility`:
  - `private` = רק היוצר/ת
  - `group` = כל מי שחבר/ה בקבוצת המשפחה
- RLS (הרשאות) מופעלות ב-Supabase כדי לאכוף את זה בצד השרת.

## טיפ ל-UX
במסך הוספה אפשר לבחור: "לשתף עם המשפחה" כדי שהרשומה תופיע בפיד המשפחתי.

בהצלחה! אם תרצי, אפשר להוסיף גם תגובות/לייקים וסטטיסטיקות.
