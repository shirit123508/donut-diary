# 📖 מדריך שימוש - יומן סופגניות

## 🖼️ איך להוסיף תמונות לסופגניות

### שלב 1: העלה תמונה לאינטרנט

בחרי אחת מהאפשרויות:

#### **אופציה 1: Imgur (הכי פשוט!)**
1. לכי ל-https://imgur.com
2. לחצי "New post"
3. העלי תמונה
4. לחצי ימין על התמונה → "Copy image address"
5. השתמשי בקישור הזה!

#### **אופציה 2: Cloudinary (מקצועי)**
1. לכי ל-https://cloudinary.com
2. צרי חשבון חינמי
3. העלי תמונה
4. העתיקי את ה-URL

#### **אופציה 3: Unsplash (תמונות מוכנות)**
```
https://images.unsplash.com/photo-1514517521153-1be72277b32f
https://images.unsplash.com/photo-1551024601-bec78aea704b
https://images.unsplash.com/photo-1551024506-0bccd828d307
```

### שלב 2: הוסיפי את הקישור

1. לכי ל-"הוספה" בניווט
2. מלאי את הפרטים
3. בשדה "קישור לתמונה" - **הדבקי את הקישור**
4. לחצי "שמירה"

**דוגמה לקישור תקין:**
```
https://i.imgur.com/abc123.jpg
https://images.unsplash.com/photo-1234567890
```

## 👨‍👩‍👧‍👦 איך לנהל משפחות

### יצירת משפחה חדשה

1. לכי ל-**"משפחה"** בניווט
2. במסך "יצירת משפחה":
   - הקלידי שם למשפחה (למשל: "משפחת כהן")
   - לחצי "יצירה"
3. **קוד הצטרפות** יופיע משמאל!
4. שתפי את הקוד עם בני המשפחה

### הצטרפות למשפחה קיימת

1. לכי ל-**"משפחה"**
2. במסך "הצטרפות עם קוד":
   - הקלידי את הקוד (6 אותיות)
   - לחצי "הצטרפות"
3. עכשיו את חלק מהמשפחה!

### יציאה ממשפחה

1. לכי ל-**"משפחה"**
2. בחרי את המשפחה מהרשימה (למעלה)
3. המשפחה הפעילה תופיע משמאל
4. לחצי על כפתור **"יציאה"** (אדום)
5. אשרי שאת רוצה לעזוב

### שיתוף סופגניות עם המשפחה

כשאת מוסיפה סופגנייה:
1. לכי ל-"הוספה"
2. מלאי את הפרטים
3. בחרי **"משפחתי (שיתוף)"** במקום "פרטי"
4. בחרי את המשפחה מהרשימה
5. שמרי - כל המשפחה תראה! 🎉

## 🔍 פתרון בעיות נפוצות

### ❌ לא רואה תמונות

**בעיה:** הוספתי קישור אבל התמונה לא מופיעה

**פתרונות:**
1. **בדקי שהקישור מסתיים ב**:`.jpg`, `.png`, `.jpeg`, `.webp`
2. **נסי לפתוח את הקישור בטאב חדש** - האם התמונה נטענת?
3. **וודאי שהקישור מתחיל ב**: `https://`
4. **דוגמה לקישור תקין**:
   ```
   ✓ https://i.imgur.com/abc123.jpg
   ✗ https://imgur.com/gallery/abc123
   ```

### ❌ המשפחה לא מופיעה אחרי יצירה

**בעיה:** יצרתי משפחה אבל היא לא מופיעה ברשימה

**פתרונות:**
1. **רעני את הדף** (F5 או Cmd+R)
2. **בדקי בדף "משפחה"** - הקוד צריך להופיע בצד ימין
3. **אם השגיאה ממשיכה** - בדקי ש-Supabase מוגדר נכון ב-`.env.local`

### ❌ כפתור העריכה לא עובד

**בעיה:** לוחצת על "עריכה" ו nothing happens

**פתרונות:**
1. **בדקי אם יש דף edit** - `/edit/[id]`
2. **נכון לעכשיו** - דף העריכה עדיין לא מוכן במלואו
3. **חלופה:** מחקי ותוסיפי מחדש

## 🔐 הגדרת הפרויקט (למפתחים)

### וידוא שפרטי Supabase נכונים

1. פתחי `.env.local`
2. וודאי שיש:
   ```env
   NEXT_PUBLIC_SUPABASE_URL=https://your-project.supabase.co
   NEXT_PUBLIC_SUPABASE_ANON_KEY=eyJhbGci...
   ```
3. קבלי את הפרטים מ-https://app.supabase.com/
   - Project Settings → API

### בדיקת חיבור

```bash
# בדקי שהשרת רץ
npm run dev

# פתחי בדפדפן
http://localhost:3000

# אם יש שגיאות - בדקי console (F12)
```
